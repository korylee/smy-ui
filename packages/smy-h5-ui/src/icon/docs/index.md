## 图标组件

> 这里为了保证图标清晰度，打包文件尽可能小，决定使用 svg 图，组件库内未使用的图片推荐使用[xions](https://www.xicons.org/)

## API

### 属性

| 参数    | 说明     | 类型               | 默认值 |
| ------- | -------- | ------------------ | ------ |
| `size`  | 尺寸     | _string \| number_ | `-`    |
| `color` | 图标颜色 | _string_           | `-`    |

### 事件

| 事件名 | 说明           | 参数           |
| ------ | -------------- | -------------- |
| `click`  | 点击图标时触发 | `event: Event` |
